# üéÖ KringleCore Enterprise

KringleCore Enterprise (v2.0) is a high-performance, aesthetically driven operations management dashboard designed for the holiday season. It features a "Zero-Click" UX, temporal state management, and smooth glassmorphism design.

## üöÄ Features

- **Temporal UI State**: Dynamically adjusts the interface based on the date:
  - **Normal View**: Standard workshop operations dashboard.
  - **Christmas Eve (Dec 24)**: Real-time Route Optimization and delivery tracking.
  - **Post-Season (Dec 30)**: Comprehensive analytics and performance review.
- **Micro-Animations**: Fluid transitions using GSAP and Framer Motion for a premium feel.
- **Interactive Splash Screen**: High-impact arrival experience with Lottie animations.
- **Glassmorphism Design**: Modern UI with frosted glass effects and neon accents.
- **Performance Optimized**: Built with TanStack Virtual for handling large datasets (e.g., gift lists) efficiently.

## üì¶ Setup & Installation

1. **Clone the repository**:
   ```bash
   git clone <repository-url>
   cd kringleCore-enterprise
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Run development server**:
   ```bash
   npm run dev
   ```
   Open [http://localhost:5173](http://localhost:5173) (or the port shown in your terminal) to view the application.
